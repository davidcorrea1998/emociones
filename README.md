# Final Project

This project is part of the IBM Coursera course final assignment.

The application detects emotions in a given text using Natural Language Processing (NLP) with Python and Flask. The system analyzes the text and identifies emotions such as joy, anger, sadness, fear, and disgust.

## Features

- Emotion detection from text input
- REST API implementation using Flask
- Error handling for invalid input
- Unit testing with unittest
- Static code analysis using pylint
- Modular package structure in Python

## Technologies Used

- Python
- Flask
- Transformers
- Hugging Face NLP model
- unittest
- pylint
